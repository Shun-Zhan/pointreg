# pointreg 环境配置 · 手把手(Windows)

你还没建 conda 环境。跟着做即可,分**建环境**和**装 GPU 依赖**两部分。
我已经把这两步写进一个批处理,先给最省事的方式,后面再给手动版备用。

---

## 方式 A:一键脚本(推荐)

### 前提
先装好 **Anaconda 或 Miniconda**(如果还没装:
https://www.anaconda.com/download 下载安装,一路默认即可)。

### 步骤
1. 在开始菜单打开 **"Anaconda Prompt"**(不是普通 CMD,普通 CMD 常找不到 conda)。
2. 进入项目目录(把路径换成你自己的):
   ```
   cd D:\pointreg--
   ```
3. 运行一键脚本:
   ```
   finetune_kit\setup_env_windows.bat
   ```
   它会自动:①建 pointreg 环境 → ②装 GPU 版 PyTorch → ③装微调依赖 → ④自检 GPU。
   中间要下载约 2GB,耐心等。

4. 看最后一行,出现 **`CUDA可用: True`** 就成功了。

装完直接接着跑微调:
```
finetune_kit\run_finetune_windows.bat
```

## 方式 B:手动一条条敲(方式 A 出问题时用)

在 **Anaconda Prompt** 里,项目根目录下依次执行:

```bat
REM 1) 创建环境(python 3.12 + open3d 等,来自 environment.yml)
conda env create -f environment.yml

REM 2) 激活
conda activate pointreg

REM 3) 装 GPU 版 PyTorch(CUDA 12.1 版本;若你显卡驱动较老见文末)
pip install torch --index-url https://download.pytorch.org/whl/cu121

REM 4) 装 GeoTransformer 微调需要的其余依赖
pip install einops coloredlogs easydict scikit-learn ipython tqdm

REM 5) 自检
python -c "import torch;print(torch.cuda.is_available())"
```
第 5 步输出 `True` 即环境就绪。

## 常见问题

**Q: 提示 `conda 不是内部或外部命令`?**
A: 你用的是普通 CMD。改用开始菜单里的 "Anaconda Prompt"。

**Q: 第 1 步提示环境已存在 / 求解失败?**
A: 先删掉重建:`conda env remove -n pointreg`,再重跑第 1 步。
   若求解很慢,可先只建 python:`conda create -n pointreg python=3.12 -y`,
   激活后 `pip install -r requirements.txt`,再做第 3、4 步。

**Q: 自检输出 `False`(CUDA 不可用)?**
A: 说明装成了 CPU 版 torch。重装 GPU 版:
   ```
   pip uninstall -y torch
   pip install torch --index-url https://download.pytorch.org/whl/cu121
   ```
   如果你的显卡驱动较老、cu121 报错,换 cu118:
   ```
   pip install torch --index-url https://download.pytorch.org/whl/cu118
   ```
   (怎么看驱动:命令行运行 `nvidia-smi`,右上角 CUDA Version 就是上限。
    ≥12.1 用 cu121,否则用 cu118。)

**Q: 只想先跑通配准、暂时不训练?**
A: 那只做方式 B 的第 1、2 步(建环境+激活)即可,GPU 依赖可以之后再装。
